# ДЗ 12: Линейный и бинарный поиск. Сортировка.

Решение домашнего задания №12.

## Содержание
1. **Задача 1**: Сравнение линейного и бинарного поиска.
2. **Задача 2**: Сортировка пузырьком vs встроенная сортировка.
3. **Задача 3**: Поиск студента.

## Как запустить
Запустите файл `main.py` для выполнения всех задач:
```bash
python main.py
```

Или запустите отдельные файлы для конкретных задач:
- `python task1.py`
- `python task2.py`
- `python task3.py`

## Описание задач

### Задача 1: Сравнение поисков
- Создается список из 1 000 000 случайных чисел.
- Ищется одно число линейным поиском.
- Список сортируется.
- Ищется то же число бинарным поиском.
- Сравнивается время выполнения обоих методов.

### Задача 2: Своя сортировка vs встроенная
- Реализована сортировка пузырьком.
- Сортируется список из 10 000 чисел пузырьком.
- Тот же список сортируется встроенной сортировкой.
- Сравнивается время выполнения.

### Задача 3: Поиск студента
- Реализована система хранения студентов (имя, группа, средний балл).
- Поиск по имени (линейный).
- Поиск по среднему баллу (бинарный, после сортировки).
- Выводится время выполнения каждого поиска.
